OK, because I'm apparently made of money - but mostly because the current process was untenable and it was unreasonable to keep expecting KerrTexas to keep doing what he was doing - I've purchased and installed an add-in called FoolBotHoneyPot.

AnyApi - Use Any Api On Registration (anti spam, anti fraud, any api thing) | Surrey Forum - The free online Surrey community

It's got a number of features that are supposed to stop bots cold, including hidden form fields. I experienced this first-hand when registering to the owner's forums. I'm used to tabbing through fields when filling out a form and I noticed that when I hit tab, I was going to fields that I couldn't see. So I guess that works. There's also "Validate IP Addresses Against StopBotters" and "Validate Email Addresses Against StopBotters" settings. So KerrTexas won't have to do this manually any more. We should see an immediate improvement. If that's the case, we can re-enable self-registration. We'll see in about 24-48 hours.

You can find all the settings in the admin panel, in Home > Options > TAC FoolBotHoneyPot (all the way at the bottom).
